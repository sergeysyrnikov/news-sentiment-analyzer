# News Sentiment Analyzer

Микросервис для анализа новостного фона компаний. 
Сервис собирает и обрабатывает новости, выполняет кластеризацию по компаниям, локациям и индустриям, а также анализирует тональность текста для выявления позитивного или негативного тренда (новостного фона).

Структура проекта построена по стандартам **Clean Architecture** (слои: Domain, Application, Infrastructure, Interfaces).

## Структура репозитория

- `app/domain/` — бизнес-модели (`NewsItem`, `Cluster`, `SentimentResult`) и порты.
- `app/application/` — бизнес-логика (сервисы кластеризации и анализа тональности).
- `app/interfaces/` — точки входа (API, CLI).
- `app/infrastructure/` — реализация работы с базой данных (в перспективе).
- `data/sample_news.json` — тестовые данные для демонстрации работы алгоритмов.
- `tests/unit/` — unit-тесты (на `pytest`).
- `docs/` — документация по архитектуре и расчетам мощностей.
- `prompts/` — заготовки промптов для работы с LLM-агентами (troubleshooting).

## Стек технологий

- Python 3.12+ (проект сконфигурирован на >=3.14.0)
- FastAPI (для построения REST API)
- PostgreSQL (целевая база данных для транзакционных данных и поиска)
- Управление зависимостями: `uv` (конфигурация через `pyproject.toml`)
- Библиотеки: `pydantic`, `rapidfuzz` (для fuzzy-match кластеризации), `pytest`

*(Примечание: На этапе PoC (MVP) проект запускается локально без Docker, полноценный HTTP API и парсеры эмулируются демо-скриптом.)*

## Установка и запуск

Проект использует `uv` для управления окружением и зависимостями.

1. **Создание окружения и установка зависимостей**:
   ```bash
   uv venv
   uv sync
   ```
   (Или установка конкретных пакетов при необходимости: `uv add rapidfuzz pydantic pytest`)

2. **Запуск демонстрационного скрипта (CLI)**:
   Демо-скрипт загружает новости из `data/sample_news.json`, выполняет их кластеризацию и выдает топ новостей по тональности.
   ```bash
   uv run python -m app.interfaces.cli.demo
   ```

3. **Запуск тестов**:
   Unit-тесты проверяют корректность нормализации, нечеткого поиска, а также оценки сентимента.
   ```bash
   uv run pytest
   ```
